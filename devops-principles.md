# DevOps Principles in Practice

This project was explicitly designed to demonstrate core DevOps principles through practical implementation rather than theory.

---

## Deployment
- Container-based deployments
- Environment-specific configuration via variables
- Image-based rollbacks supported

## Maintainability
- Modular Flask Blueprints
- Clear repository structure
- Strong documentation focus

## Security
- Argon2 password hashing
- Rate limiting on sensitive routes
- SSL-enforced database connections
- Configuration isolation via environment variables

## Observability
- Structured application logging
- Persistent audit logs stored in the database
- Azure App Service logging enabled

## Scalability
- Stateless application container
- External managed database
- Azure-supported horizontal scaling

## Continuous Integration
- Automated image builds via GitHub Actions
- Reproducible dependency installation
