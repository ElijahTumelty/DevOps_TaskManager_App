# Lessons Learned

This project provided hands-on experience with real-world DevOps challenges.

---

## Key Takeaways

- Containerisation simplifies deployment consistency
- Managed cloud services reduce operational complexity
- Documentation is critical for maintainability
- Security must be integrated early, not added later

---

## Challenges Faced

- Cloud database connectivity
- Container limitations on student subscriptions
- Adapting architecture under deployment constraints

---

## Future Improvements

- CI pipelines with automated testing
- Production-grade WSGI server (Gunicorn)
- Advanced observability tooling
- Kubernetes-based orchestration

---

## Final Reflection

This project strengthened my confidence in designing, deploying, and reasoning about DevOps-oriented systems and reflects a practical understanding of modern cloud-native engineering.
